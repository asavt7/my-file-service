# FILE SERVICE
simple file service study project with aws s3 storage

### Features
- [x] uploading files
- [x] downloading files

### Stack:
- golang
- docker/docker-compose
- minio (for local development)
- aws s3
- swagger

## How to use

Run
```shell
make docker-compose-up
```
And open [swagger ui](http://localhost:8080/swagger/index.html)
